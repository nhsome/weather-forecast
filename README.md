# Weather forecast

## Backend

**Requirements:**

1. node.js v12.16.1
2. npm v6.13.4
3. pm2 installed globally


**Firstly:** 

	 copy ecosystem.example.config.js to ecosystem.config.js and edit them

**Run in dev mode:** 
	 
	 $ npm run dev

**Start in production mode:** 
	 
	 $ npm run start

## Frontend

**Requirements:**

1. node.js v12.16.1
2. npm v6.13.4


**Run in dev mode:** 
	 
	 $ npm start

**Build for production:** 
	 
	 $ npm run build
