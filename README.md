# Weather forecast

## Backend

**Requirements:**

1. node.js v12.16.1
2. npm v6.13.4
3. pm2 installed globally (`$ npm install pm2 -g`)


**Firstly:** 

- `$ npm install`

- `$ cp ecosystem.example.config.js ecosystem.config.js`

- Edit ecosystem.config.js

**Run in dev mode:** 
	 
`$ npm run dev`

**Start in production mode:** 
	 
`$ npm run start`

## Frontend

**Requirements:**

1. node.js v12.16.1
2. npm v6.13.4

**Firstly:** 

- `$ cd frontend/`
- `$ npm install`

**Run in dev mode:** 
	 
`$ npm start`

**Build for production:** 
	 
`$ npm run build`
